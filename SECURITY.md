# Security Policy

## Supported Versions
Only the latest major version of LLMHub is actively supported for security updates.

## Reporting a Vulnerability
Please **do not** report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in LLMHub, please email the maintainers at security@llmhub.example.com. We will respond within 48 hours and work with you to resolve the issue before public disclosure.
