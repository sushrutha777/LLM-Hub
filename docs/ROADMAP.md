# LLMHub Roadmap

This document outlines the strategic roadmap for LLMHub. It is divided into 5 phases to ensure stable, iterative growth.

## Phase 1: Authentication & Playground (Current)
- [x] Open-Source Repository Scaffolding
- [x] Basic Auth Service (JWT)
- [ ] API Gateway Routing
- [ ] Model Router scaffolding
- [ ] Ollama Provider Plugin
- [ ] Basic React Playground

## Phase 2: Multi-Model Routing & Cache
- [ ] Provider Plugins (OpenAI, vLLM, Gemini, Anthropic, Groq)
- [ ] Semantic & Exact Caching via Redis
- [ ] Analytics Service (Tokens, Latency)
- [ ] Multi-Model UI Comparison

## Phase 3: Control Plane & Docker
- [ ] Admin Control Plane APIs
- [ ] Prometheus & Grafana Monitoring Integrations
- [ ] Docker Compose zero-config setup

## Phase 4: Kubernetes & Plugins
- [ ] Kubernetes Manifests (Deployments, HPA, Ingress)
- [ ] Formalized Plugin System Extensibility
- [ ] Python SDK
- [ ] JS/TS SDK

## Phase 5: Enterprise
- [ ] Multi-Tenant Workspaces
- [ ] Enterprise RBAC (Role-Based Access Control)
- [ ] Helm Charts for K8s deployment
